# XCode-Templates

XCode-Templates is a set of class templates to use in XCode. 

# Available
1. BDD</br>
   * Quick/Nimble (Swift)</br>
   * Specta/Expecta (Objective-C)</br>

# Install

```
./install.sh
```

# Uninstall

```
./uninstall.sh
```
